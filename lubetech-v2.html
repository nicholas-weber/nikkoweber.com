Lube-Tech V2

lubetech.com

Lube-Tech V2 was the first enterprise-level web application that I ever worked on professionally. It contains both a marketing and eCommerce portion and was built in Angular 2+ and uses lazy-loaded components as faux "pages."

  • In charge of UX design, with input from corporate and marketing
  • Each page includes its own SCSS file with a further hierarchy of SCSS above it
  • CSS Flexbox used extensively for both complex layouts and responsiveness
  • Built a stylish, horizontally-oriented navigation system with multiple columns and subheaders
  • Created a custom HTML5/CSS accordion to make pages more readable
  • Responsible for the decision to use a monochromatic palette, in addition to Lube-Tech's signature color
  • In charge of SEO for marketing pages
  • Integrated a Google Font similar to the company's official marketing font (Titillium Web)
  • Became comfortable with using Git as a team
  • Continued to work with marketing to integrate large and small enhancements
